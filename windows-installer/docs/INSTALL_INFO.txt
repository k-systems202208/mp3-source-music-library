自宅音楽ライブラリ v2.6.2
インストール前のご確認
================================

このインストーラーは、
自宅音楽ライブラリ v2.6.2をWindowsへ導入します。

利用者が選択したMP3フォルダを読み取り、
ブラウザから検索・再生できるローカル音楽ライブラリを作成します。

音源の扱い
----------
・MP3は別の場所へコピーしません。
・利用者が選択した既存フォルダをそのまま参照します。
・MP3音源はインストーラーに含まれていません。

管理データ
----------
次へ保存します。

%LOCALAPPDATA%\MusicLibrary

主な内容:
・library.db
・再生回数
・表記補正
・アートワークキャッシュ
・バックアップ
・ログ
・外部接続URL

外部接続
--------
・Tailscaleは公式サイトから別途インストールします。
・管理画面の「外部接続をかんたん設定」から案内されます。
・Tailscaleへのログインは利用者本人が行います。
・ルーターのポート開放は使用しません。
・Tailscale Funnelは使用しません。

正しい外部URL形式:

https://PC名.tailnet名.ts.net/music-library-search.html

更新
----
従来版がインストール済みの場合は、
通常アンインストールせずv2.6.2を上書きインストールできます。
利用者データは維持されます。
