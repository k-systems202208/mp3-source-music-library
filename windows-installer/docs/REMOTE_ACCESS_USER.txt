自宅音楽ライブラリ v2.6.2
外部接続ガイド
================================

v2.6.2では、管理画面の
「外部接続をかんたん設定」から
Tailscale Serveを設定できます。

利用者の操作
------------
1. 自宅音楽ライブラリ v2.6.2を起動する
2. 「外部接続をかんたん設定」を押す
3. Tailscale未導入時は公式ページからインストールする
4. Windows右下のTailscaleアイコンからログインする
5. もう一度「外部接続をかんたん設定」を押す
6. 表示された完全なURLをスマートフォンで開く

正しいURL:

https://PC名.tailnet名.ts.net/music-library-search.html

v2.6.2では、URL末尾の
/music-library-search.html
が自動的に補完されます。

スマートフォン
--------------
・iPhoneまたはAndroidへTailscaleをインストールします。
・自宅PCと同じTailscaleアカウントでログインします。
・Wi-Fiを切り、4Gまたは5Gでも再生できるか確認します。

家族など別アカウント
--------------------
Tailscale管理画面で自宅PCを相手へShareします。
同じアカウントのパスワードを共有しないでください。

詳細はREMOTE_ACCESS_FAMILY.txtを参照してください。

利用条件
--------
・自宅PCの電源が入っている
・PCがスリープしていない
・Tailscaleが接続中
・自宅音楽ライブラリ v2.6.2が起動中

外部接続を停止
--------------
管理画面の「外部接続を停止」を押します。
PC内のローカル利用は停止しません。

安全上の注意
------------
・Tailscale Funnelは使用しません。
・ルーターのポート開放は不要です。
・音楽ライブラリ本体は127.0.0.1だけで待ち受けます。
・Tailscaleの認証は自動化しません。
・接続URLを一般公開しないでください。
