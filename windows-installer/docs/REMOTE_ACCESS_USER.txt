自宅音楽ライブラリ 外部接続ガイド
====================================

このバージョンでは、管理画面の「外部接続をかんたん設定」から
Tailscale Serveを設定できます。

利用者の操作
------------
1. 自宅音楽ライブラリを起動する
2. 「外部接続をかんたん設定」を押す
3. Tailscaleが未導入なら、開いた公式ページからインストールする
4. Windows右下のTailscaleアイコンからログインする
5. もう一度「外部接続をかんたん設定」を押す
6. 表示された https://～.ts.net/music-library-search.html のURLをスマートフォンで開く

スマートフォン
--------------
・iPhone／AndroidへTailscaleアプリをインストールします。
・自宅PCと同じTailscaleアカウントでログインします。
・Wi-Fiを切り、4G／5G回線でも曲を再生できるか確認します。

家族など別アカウント
--------------------
Tailscaleの管理画面で、自宅PCを相手へShareします。
同じアカウントのパスワードを共有しないでください。
詳細はREMOTE_ACCESS_FAMILY.txtを参照してください。

必要な条件
----------
・自宅PCの電源が入っている
・PCがスリープしていない
・Tailscaleが接続中
・自宅音楽ライブラリの管理画面が起動中

外部接続を停止
--------------
管理画面の「外部接続を停止」を押します。
PC内のローカル利用は停止しません。

安全上の注意
------------
・Tailscale Funnelは使いません。
・ルーターのポート開放は不要です。
・音楽ライブラリ本体は127.0.0.1だけで待ち受けます。
・Tailscaleのインストールとアカウント認証は自動化できません。
・接続URLを一般公開しないでください。
