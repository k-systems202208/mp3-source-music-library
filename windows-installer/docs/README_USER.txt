自宅音楽ライブラリ 利用ガイド
================================

1. インストール
---------------
MusicLibrary-Setup-2.6.1-x64.exeをダブルクリックします。
管理者権限は通常不要です。

2. 初回起動
-----------
スタートメニューの「自宅音楽ライブラリ」を開きます。
MP3ファイルが保存されているフォルダを選択します。
MP3は別の場所へコピーされません。

3. ローカル利用
---------------
「ライブラリを開始」を押すと、MP3の追加・削除・変更を確認した後、
ブラウザが自動で開きます。

4. 外部接続
-----------
管理画面の「外部接続をかんたん設定」を押します。
Tailscaleのインストールとログインだけは、利用者本人の操作が必要です。
ログイン後は、Tailscale Serveの設定、URL取得、URL保存をアプリが行います。

スタートメニューの「外部接続を設定」からも開始できます。
詳細はREMOTE_ACCESS_USER.txtを参照してください。

5. 保存場所
-----------
管理情報は次へ保存されます。

%LOCALAPPDATA%\MusicLibrary

主な内容:
・library.db
・Backups
・Exports
・.artwork-cache
・Logs
・config.json
・remote-url.txt

6. 更新
-------
新しいインストーラーを現在のアプリへ上書きインストールします。
library.db、設定、外部接続URLは維持されます。

7. アンインストール
-------------------
Windowsの「インストールされているアプリ」から削除できます。
利用者データは安全のため残ります。

8. 注意
-------
・PCがスリープまたは電源オフの時は外部から利用できません。
・Tailscale Funnelやルーターのポート開放は使用しません。
・市販MP3の配布機能ではありません。
