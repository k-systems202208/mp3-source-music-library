自宅音楽ライブラリ v2.6.2
利用ガイド
================================

1. インストール
---------------
MusicLibrary-Setup-2.6.2-x64.exeをダブルクリックします。
管理者権限は通常不要です。

2. 初回起動
-----------
スタートメニューの「自宅音楽ライブラリ」を開きます。
MP3ファイルが保存されているフォルダを選択します。
MP3は別の場所へコピーされません。

3. ローカル利用
---------------
「ライブラリを開始」を押すと、
MP3の追加・削除・変更を確認した後、
ブラウザが自動で開きます。

ローカルURL:

http://127.0.0.1:8765/music-library-search.html

4. 外部接続
-----------
管理画面の「外部接続をかんたん設定」を押します。

Tailscaleのインストールとログインは、
利用者本人の操作が必要です。

ログイン後は、
Tailscale Serve設定、URL取得、URL保存をアプリが行います。

正しい外部URL:

https://PC名.tailnet名.ts.net/music-library-search.html

スタートメニューの「外部接続を設定」からも開始できます。
詳細はREMOTE_ACCESS_USER.txtを参照してください。

5. 保存場所
-----------
%LOCALAPPDATA%\MusicLibrary

主な内容:
・library.db
・Backups
・Exports
・.artwork-cache
・Logs
・config.json
・remote-url.txt

6. v2.6.2への更新
-----------------
新しいv2.6.2インストーラーを、
現在のアプリへ上書きインストールします。

library.db、設定、再生回数、表記補正、
外部接続URLは維持されます。

7. アンインストール
-------------------
Windowsの「インストールされているアプリ」から削除できます。
利用者データは安全のため残ります。

8. 注意
-------
・PCがスリープまたは電源オフの時は外部利用できません。
・Tailscale Funnelは使用しません。
・ルーターのポート開放は使用しません。
・市販MP3を第三者へ配布する機能ではありません。
