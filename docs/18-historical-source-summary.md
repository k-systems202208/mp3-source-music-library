# 現行チャット以前の作業記録要約

## 目的

利用者提供の過去作業記録を、公開可能な形へ要約したものです。会話ログそのものではなく、要件と設計判断だけを整理しています。

## 要約

1. iTunes XMLを検索するWebアプリとして開始
2. 8,383曲を単一HTMLへ埋め込み
3. サーバー・DBなし、無料・オフライン利用を実現
4. カード目録風UI
5. 曲カードを曲名・アルバム・アーティストへ簡素化
6. アルバム・アーティストをリンク化
7. フィルターチップと範囲内検索
8. ローマ字タイトルの自動日本語変換は不採用
9. 編集・Google確認・英数字タイトル・訂正済みを追加
10. アーティスト名の一括補正
11. プルダウンを廃止して3ビュー化
12. アーティスト→アルバム→曲のドリルダウン
13. パンくず追加
14. JSON外部化へ移行

## 現行版への継承

| 初期概念 | 現行実装 |
|---|---|
| カード目録 | 曲・グループカード |
| 3ビュー | songs / artists / albums |
| ドリルダウン | SQLite APIのview |
| パンくず | ブラウザ状態から生成 |
| 曲名補正 | tracks.title_override |
| アーティスト補正 | artists.display_name_override |
| 英数字のみ | SQLite独自関数 |
| 訂正済み | overrideの有無 |
| Google確認 | 曲カードの検索リンク |
| 追加読み込み | 80件APIページング |
| 外部JSON | SQLite APIへ発展 |
