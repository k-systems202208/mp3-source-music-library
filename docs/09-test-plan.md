## 7. セキュリティテスト

ブラウザから以下が404になること:

```text
/library.db
/database.py
/start-music-library.bat
/Backups/
/Exports/
/legacy-library-data.json
```

確認:

- POST本文65KiB超で拒否
- 不正viewで400
- 不正indexKeyで400
- 存在しないtrackで404
- HTML/APIがno-store
- MP3は取得可能
