# スクリーンショット撮影チェック

公開用の画像は、実利用環境ではなくダミー音源で撮影することを推奨します。

## 推奨カット

1. 曲一覧
2. 曲名索引
3. アーティスト索引
4. アルバム一覧
5. 下部プレーヤー
6. 最大化プレーヤー
7. スマホ表示
8. アーキテクチャ図

## マスク対象

- Windowsユーザー名
- フルパス
- IPv4
- Tailscale IPv4
- `.ts.net`の実名
- tailnet名
- メールアドレス
- 家族の氏名・端末名
- 市販音源のアートワーク

## ファイル名例

```text
01-song-list.png
02-song-index.png
03-artist-index.png
04-album-list.png
05-player-modal.png
06-mobile-view.png
```
